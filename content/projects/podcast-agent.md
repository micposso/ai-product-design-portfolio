# Podcast Agent For Summarizing Popular Web Content

## Summary

An AI agent that identifies high-interest content on the web, summarizes it, and turns it into a concise ten-minute podcast using voice and text-to-speech workflows.

## Context

The idea is to create a more useful media format for people who want signal instead of noise. Instead of manually reading long threads, articles, or discussions, the agent curates topics, summarizes them, and produces an audio briefing.

## What I Built

- Defined an agent workflow that selects relevant or trending web content.
- Designed summarization steps that compress the material into a short, high-signal narrative.
- Planned voice and text-to-speech pipelines to produce a ten-minute podcast experience.
- Framed the project as a content automation workflow that combines discovery, summarization, scripting, and audio generation.

## Stack

Agent workflows, summarization pipelines, voice systems, text-to-speech, content curation, podcast automation

## Outcome

The project reflects my interest in AI-powered media workflows and voice interfaces. It combines agent orchestration with practical content design for an everyday listening experience.
